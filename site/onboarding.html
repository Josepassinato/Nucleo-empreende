passo 2 de 6

Instalar o framework

Cole o comando abaixo no terminal do seu servidor. O instalador faz todo o resto automaticamente.

💡
Se você está em uma VPS, primeiro conecte via SSH:
ssh ubuntu@IP_DO_SEU_SERVIDOR
TERMINAL 
# Cole no terminal e pressione Enter
curl -fsSL https://install.nucleoempreende.com.br | bash -s SUA_CHAVE_AQUI
🔑
Substitua SUA_CHAVE_AQUI pela chave de licença que você recebeu no e-mail de compra.
Formato: NF-XXXX-XXXX-XXXX-XXXX
📋 O que o instalador faz
✓ Verifica Python 3.10+
✓ Valida sua licença
✓ Instala 25+ dependências
✓ Configura Redis
✓ Cria o comando nucleo
✓ Cria serviço systemd
✓ Abre o Setup Wizard
✓ Testa conexões
⏳ Quanto tempo demora?
Download do framework~30 segundos
Instalação das dependências Python~3–5 minutos
Playwright (Chromium)~2 minutos
Total~5–8 minutos

passo 4 de 6

Testar os conectores

Antes de iniciar em produção, vamos verificar que tudo está funcionando. Rode o comando abaixo no terminal do servidor.

TERMINAL 
cd ~/nucleo-empreende
python3 testar_tudo.py
Simulação da saída esperada
Clique em "Simular" para ver como deve aparecer no seu terminal →
🔍 Interpretando o resultado
✅ produçãoAPI key configurada e conexão OK
⚠️ simulaçãoSem API key — funciona mas não faz chamadas reais
❌ erroChave configurada mas inválida — verifique o formato
⚠️
Para continuar, você precisa de pelo menos Gemini, Groq e Twilio em modo produção. Os demais podem ficar em simulação.
passo 5 de 6

Iniciar os agentes

Tudo validado. É hora de ligar a diretoria.

TERMINAL 
nucleo start
🧠 O que acontece agora
01

Agentes inicializam

Cada agente carrega sua personalidade, memória e lista de tarefas. Leva ~30 segundos.

02

Lucas faz um briefing

O CEO lê o estado da empresa (configurações, dados disponíveis) e distribui as primeiras tarefas para a equipe.

03

Você recebe uma mensagem no WhatsApp

A Ana Costa (CHRO) envia uma mensagem te apresentando a equipe e confirmando que todos estão operando.

04

Primeira reunião em 24h

O sistema agenda a primeira reunião semanal. Você recebe a ata por e-mail e WhatsApp.

Acessar o dashboard

O painel de controle mostra o estado de todos os agentes, aprovações pendentes e atividade em tempo real.

TERMINAL 
nucleo dashboard

# Dashboard disponível em:
http://localhost:8000   # local
http://IP_DO_SERVIDOR:8000   # VPS
📊 O que você vê no dashboard
🧠 Estado dos agentes
Score, estresse, energia e tarefa atual de cada agente em tempo real via WebSocket
💰 Aprovações pendentes
Gastos acima do limite aguardando sua aprovação com 1 clique
📋 Feed de atividade
Tudo que os agentes fizeram nas últimas horas

Latência, chamadas e custo de cada integração no dia
